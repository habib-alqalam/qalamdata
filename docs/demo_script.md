# Demo Script (1 to 3 Minutes)

## Goal
Demonstrate problem clarity, execution quality, AI value, and impact in under 3 minutes.

## Script

1. Opening (20 seconds)
- "This solution helps PropTech teams make faster parcel-level investment decisions using explainable AI scoring."

2. Problem (20 seconds)
- "Teams often have fragmented parcel data and unclear prioritization. This app creates instant, explainable recommendations."

3. Live App Walkthrough (80 seconds)
- Open dashboard.
- Show problem summary section.
- Enter sample parcel values.
- Click Analyze.
- Explain score, confidence, and recommendation.
- Open impact model endpoint output in UI.

4. Architecture and AI (30 seconds)
- "FastAPI powers modular scoring logic and transparent explanations. The design supports future model upgrades."

5. Impact (20 seconds)
- "The result is faster screening, clearer decisions, and improved investment quality for real estate stakeholders."

6. Close (10 seconds)
- "Repository, live app, and this demo are linked in the submission issue form."
