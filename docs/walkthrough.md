# Walkthrough For Judges

## What You Will See

A simple dashboard where you can:
- Understand the problem quickly
- Run parcel analysis in one form
- See AI-backed score, confidence, and recommendations
- Understand impact outcomes in plain language

## 60-Second Judge Path

1. Open the live app link.
2. Read the top section: problem and value.
3. Enter sample parcel inputs.
4. Click Analyze.
5. Review score, confidence, rationale, and impact model.

## Why It Is Self-Explanatory

- Every result card includes a short explanation
- Labels avoid technical jargon
- Inputs include placeholder examples
- Results map to judging criteria directly

## Sample Input

- parcel_id: 9140347
- is_registered: true
- is_free_hold: false
- building_density: 0.64
- transit_access_score: 0.72
- estimated_demand_index: 0.81

## Expected Output Snapshot

- opportunity_score: 74
- confidence: high
- recommendation: prioritize compliance optimization and premium tenant targeting
- impact_summary: faster decision cycle and improved portfolio yield potential
