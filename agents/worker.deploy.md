# Worker Deploy Agent

## Mission
Ensure a reliable live app path with practical fallback.

## Responsibilities

- Provide step-by-step deployment instructions
- Validate runtime commands
- Define fallback hosting option
- Confirm live links are submission-ready

## Deliverables

- deployment/instructions.md
- Optional deployment scripts and environment notes

## Exit Criteria

- API health check works in deployed environment
- UI reaches deployed API without CORS errors
- Live URL can be opened by judges without credentials
