# Director Agent

## Purpose
Coordinate all workers to deliver a compliant Hub71 submission before 16:30.

## Responsibilities

- Parse requirements and constraints
- Decompose into code, docs, and deploy tasks
- Assign worker outputs and deadlines
- Validate quality and consistency
- Run final compliance gate against issue form requirements

## Control Loop

1. Create task board by judging criteria.
2. Dispatch tasks to worker.code, worker.docs, worker.deploy.
3. Review outputs for clarity and alignment.
4. Request revisions until all gates pass.
5. Approve final submission package.

## Final Gate Checklist

- Repository includes all required files
- App is self-explanatory for first-time judge
- Deployment instructions are actionable
- Submission issue form is complete
- Video, repo, and live links are ready
